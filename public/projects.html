Web Development & Full-Stack Engineering

Interactive AI Resume Portfolio

Overview: Designed and built this interactive resume as a full-stack web application. The site features a conversational AI assistant powered by the OpenAI API that answers recruiter and employer questions about my background, skills, and experience in real time.

My Role: Sole developer — responsible for architecture decisions, backend API integration, frontend design, structured data modeling, and deployment.

Challenges: The biggest challenge was structuring my personal data in a way the AI could reference accurately without hallucinating. This required designing a modular JSON knowledge base covering topics like leadership, ethics, performance, and career goals, and building a context-injection system in the server-side API handler.

Skills Gained: This project sharpened my ability to think like a systems architect — how data flows from a data layer through business logic to a user interface. I also developed practical experience with REST API integration, prompt engineering, and Node.js server design. Professionally, it taught me how to communicate technical work to a non-technical audience, which is a core consulting skill.

Node.js JavaScript OpenAI API REST APIs HTML/CSS Systems Design Prompt Engineering

User Authentication & News Aggregation Platform

Overview: Built a multi-user web platform with secure login and registration, a role-based dashboard, real-time chat via WebSockets, and a news aggregation service that pulls and categorizes articles from external sources. The system handles session management, user threads, and audit logging.

My Role: Full-stack developer — designed the database schema, implemented authentication middleware, wrote the news aggregator service, and structured the socket-based chat system.

Challenges: Security was the primary challenge. Implementing proper authentication required careful attention to password hashing, session tokens, input validation, and protecting routes from unauthorized access. Building the SSRF-prevention middleware added another layer of complexity, requiring me to research and apply OWASP security principles directly in code.

Skills Gained: This project gave me hands-on experience with real-world security concerns — not just in theory but in implementation. I learned how to structure middleware pipelines, design relational data models across multiple entities, and reason about trust boundaries in a web system. These are skills directly applicable to IT consulting, where I will advise clients on secure system design and risk mitigation.

Node.js Express.js WebSockets Authentication & Authorization Database Design Security (OWASP) Middleware Architecture
